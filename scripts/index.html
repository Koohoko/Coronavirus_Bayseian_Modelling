Bayesian Modelling on Codon Usage Among Different Virus Genus of Coronavirus

Haogao Gu

14/08/2019

Understand the problem

Codon usage is a famous genomic signature for many RNA viruses, including Coronavirus. In this project, we planned to investigate the distributions of the Relative Synonymous Codon Usage (rscu) of a specific codon in coronavirus of differnt virus genes, by bayesian hierarchical modelling.

Collection of relevant data

The CDS fasta sequences were retrieved from public database. Necessary data cleaning were performed (details not shown).

Explore the data

After some initial data cleaning, we get the data suitable for fitting a model. First we would take a overview of the data.

Distributions of the codon usage by different virus genus

We use the Relative Synonymous Codon Usage (RSCU) as a method to estimate the codon usage bias.

Postulate a model

We assumed that And the RSCU values for each group come from individual gamma distribution with its own alpha and beta coefficients. And the alpha and beta of the distributions for different groups (virus genuses) come from two different normal distributions with different mean and variance. Thus we can build a hierarchical model basing on these data.

Fit the model

library(rjags)
df_jags <- subset(df_tmp, codon == "TTT")
df_jags$group_values <- as.numeric(factor(df_jags$`Virus Genus`))

mod_string = " model {
for (i in 1:length(RSCU)) {
  RSCU[i] ~ dgamma(alpha[group_values[i]], beta[group_values[i]])
}

for (j in 1:max(group_values)) {
  alpha[j] ~ dnorm(theta1, 1/sigma1)
  beta[j] ~ dnorm(theta2, 1/sigma2)
}

theta1 ~ dnorm(1.0, 1/100)
theta2 ~ dnorm(1.0, 1/100)

sigma1 ~ dexp(1.0)
sigma2 ~ dexp(1.0)

} "

set.seed(2019)

data_jags = as.list(df_jags)

params = c("alpha", "beta", "theta1", "theta2", "sigma1", "sigma2")

mod = jags.model(textConnection(mod_string), data=data_jags, n.chains=3)
update(mod, 1e3)

mod_sim = coda.samples(model=mod,
                       variable.names=params,
                       n.iter=5e5)
mod_csim = as.mcmc(do.call(rbind, mod_sim))

## convergence diagnostics
plot(mod_sim)

gelman.diag(mod_sim)
autocorr.diag(mod_sim)
autocorr.plot(mod_sim)
effectiveSize(mod_sim)

Use the model

we can check the posterior probability that the RSCU values for the

Alphacoronavirus is greater than the RSCU values for the Betacoronavirus

n_sim <- nrow(mod_csim)
set.seed(2019)
alpha_rscu_pred <- rgamma(n_sim,mod_csim[,1], mod_csim[,5])
beta_rscu_pred <- rgamma(n_sim,mod_csim[,2], mod_csim[,6])

mean(alpha_rscu_pred > beta_rscu_pred)
## [1] 0.3844247

The posterior probability is 0.384.
